up arrow: its approve the mask generated by sam3, so the name of the image will be saved in the output csv as "approved".

down arrow: its disapprove the mask generated by sam3, so the name of the image will be saved in the output csv as "disapproved".

Right arrow: Nothing

change to following options:

up arrow (Segmentation): the mask generated by sam3 is approved, so the name of the image will be saved in the output csv as "segmentation".

down arrow (Trash): the mask generated by sam3 is disapproved and the orginal annotation is bad, so we cant use this image. Save the output as "nothing".

Right arrow (Detection): the mask generated by sam3 is disapproved but the orginal annotation is good, so we can use the original annotation as ground truth. Save the output as "detection".

Left arrow (Return Later): keep the same


So the new output of the csv file will be like this:

```csv
image_name,decision
image_name1.jpg,segmentation
image_name2.jpg,nothing
image_name3.jpg,detection
```
